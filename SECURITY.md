# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.2   | :white_check_mark: |
| 0.2.1   | :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Porcupin, please report it to **info@fafolab.xyz**.

We will respond within **3 business days** to confirm receipt of your report.

### Responsible Disclosure

We ask that all security disclosures be made responsibly:

-   Please provide appropriate time for us to remediate security concerns before making them public
-   We welcome pull requests with remediations if you are able to provide them

Thank you for helping keep Porcupin secure.
